# 第 1 周 Day 2：文本文件统计：行数、词数、字符数

日期：2026-05-26（周二）

本周主题：环境搭建与 Python 文本处理起步

今日目标：写出第一个能处理真实文本文件的 Python 练习。

预计投入：2-3 小时。如果当天时间少，优先完成“作业要求”的前 3 项，并在学习日志里说明延期项。

## 作业要求

1. 准备一个测试文本文件，至少包含中文、英文、空行各一部分。
2. 写函数读取文件内容，统计行数、词数、字符数。
3. 把统计结果打印到终端，格式要清楚。
4. 手动测试空文件、只有一行、包含空行的文件。
5. 在学习日志里写清楚：read、split、len 分别做了什么。

- 统计结果和你手动数的结果大体一致。
