# 第 1 周 Day 1：搭好仓库和学习记录系统

日期：2026-05-25（周一）

本周主题：环境搭建与 Python 文本处理起步

今日目标：确认学习环境可用，并让每天的学习有日志、代码和追踪表证据。

预计投入：2-3 小时。如果当天时间少，优先完成“作业要求”的前 3 项，并在学习日志里说明延期项。

## 作业要求

1. 确认 Python、Git、Node.js 可以在终端运行，记录版本号。
2. 检查仓库目录：README、learning-log、python-foundations、trackers 是否能找到。
3. 填写 trackers/skill-matrix.csv 的当前水平，先诚实写 0/1/2，不追求好看。
4. 创建今天的学习日志，写下本周最小目标和今天遇到的问题。
5. 运行一个最小 Python 文件，输出一句话，确认终端和编辑器工作正常。
