主线阅读：阅读 [fastapi/full-stack-fastapi-template](https://github.com/fastapi/full-stack-fastapi-template) 的 README 和项目结构。

只看：

- README 的启动、配置、测试说明。
- 后端目录结构。

迁移点：把自己的项目 README 改到陌生人能运行的程度。

Java 后端对照：阅读 [spring-projects/spring-petclinic](https://github.com/spring-projects/spring-petclinic) 的 README。

只看：

- 项目如何说明运行方式。
- 项目如何组织 domain / web / data 层。

迁移点：给 Java + AI 后端网关写一个最小 README 目录。
