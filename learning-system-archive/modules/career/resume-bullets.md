# 简历 Bullet 模板

推荐格式：

`使用 [技术栈] 构建 [系统]，用于 [解决的问题]，并通过 [评估方式或结果] 验证效果。`

## AI 学习助手

- 使用 Python/TypeScript 构建 AI 学习助手，支持文本总结、追问回答和会话历史保存。
- 实现后端 API、结构化 LLM 输出、错误状态处理，并为核心总结逻辑编写测试。
- 通过 README、架构图和 demo 演示提升项目可展示性。

## 个人 RAG 知识库

- 使用 FastAPI、embedding 和向量数据库构建 RAG 知识库，支持带来源引用的问答。
- 实现文档切块、语义检索、基于证据的回答生成，以及 20-50 条测试问题的评估集。
- 增加日志和失败案例分析，用于定位检索失败和无依据回答。

## LangGraph 研究 Agent

- 基于 LangGraph 构建研究 Agent，支持任务拆解、工具调用、状态追踪和结构化报告生成。
- 实现工具白名单、执行日志、重试处理和高风险动作人工确认。
- 设计 Agent 执行轨迹日志，用于解释每一步推理和工具调用。
