## 5. 最小知识讲解

### 5.1 页面是组件组合，不是一个大字符串

React 组件是返回 UI 描述的函数。JSX 看起来像 HTML，但可以嵌入 JavaScript 表达式和 typed object 字段。

```tsx
function StatusLabel() {
  const completed = 3;
  return <p>已完成 {completed} 项</p>;
}
```

### 5.2 今天只关心四个区域

- 标题：告诉用户工具是什么。
- 文本输入：提供待总结内容。
- 提交按钮：后续触发行为。
- 输出区域：展示 mock 摘要。

### 5.3 为什么先用 mock

mock 把“页面是否正确展示”与“网络/API 是否正常”分开。先验证一个边界，之后出错时更容易定位。

与 Python 对照：这类似先给函数传固定测试数据，确认显示逻辑，再接真实文件或 LLM。

常见误区：页面能显示不代表交互已完成。今天先建立结构，不提前混入 Day 4–6 的状态和异步逻辑。
