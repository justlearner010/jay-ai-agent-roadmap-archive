




## 一句话解释
venv :virtual environment 
python中的虚拟环境
用自己的话解释这个概念是什么。

## 为什么重要
用于隔离每个项目之间需要的不同环境
说明它和当前项目、AI Agent、RAG、后端工程或科班基础有什么关系。
对于我们项目安全性和高效性提供保证


## 概念及代码

```python
venv :most common one
python3 -m venv env#env为环境名字

source env/bin/activate
deactive#停用
pip :prefered install program#这个指令来安装python包

pip list


pip install request

pip freeze > requirements.txt
pip install -r requirements.txt #通过requirements.txt来让pip下载所需要的python package
rm -r my_env #delete the virtual environment

```
