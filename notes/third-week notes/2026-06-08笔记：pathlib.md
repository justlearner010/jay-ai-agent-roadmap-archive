# 1. 路径拼接

不要这样写：

```
path = "data/" + filename
```

也不要：

```
path = os.path.join("data", filename)
```

直接：

```
from pathlib import Pathdata_dir = Path("data")file_path = data_dir / "article.txt"print(file_path)
```

输出：

```
data/article.txt
```

Windows 下自动变成：

```
data\article.txt
```

### 多层拼接

```
from pathlib import Pathpath = Path("output") / "2026" / "report.json"print(path)
```

结果：

```
output/2026/report.json
```

# 2. 获取文件信息

## 文件名

```
from pathlib import Pathp = Path("data/report.txt")print(p.name)
```

输出：

```
report.txt
```

---

## 文件后缀

```
print(p.suffix)
```

输出：

```
.txt
```

---

## 文件名（不含后缀）

```
print(p.stem)
```

输出：

```
report
```

---

## 父目录

```
print(p.parent)
```

输出：

```
data
```

# 4. 读文件

你之前的项目：

```
with open(fname) as f:    text = f.read()
```

可以改成：

```
from pathlib import Pathtext = Path("article.txt").read_text(    encoding="utf-8")print(text)
```

---

## 异常处理

```
from pathlib import Pathtry:    text = Path("article.txt").read_text(        encoding="utf-8"    )except FileNotFoundError:    print("文件不存在")except PermissionError:    print("没有读取权限")
```

这是生产代码更常见的写法。

# 6. 遍历目录

## 当前目录所有文件

```
from pathlib import Pathfor item in Path(".").iterdir():    print(item)
```

可能输出：

```
main.pydataREADME.md
```

---

## 只看文件

```
for item in Path(".").iterdir():    if item.is_file():        print(item)
```

---

## 只看目录

```
for item in Path(".").iterdir():    if item.is_dir():        print(item)
```

---

# 7. 查找所有 Python 文件

## 当前目录

```
from pathlib import Pathfor py_file in Path(".").glob("*.py"):    print(py_file)
```

---

## 递归查找

```
for py_file in Path(".").rglob("*.py"):    print(py_file)
```

等价于：

```
find . -name "*.py"
```

对于你的 AI Agent 项目特别常用。

例如统计项目代码行数：

```
from pathlib import Pathtotal = 0for file in Path(".").rglob("*.py"):    total += len(        file.read_text(            encoding="utf-8"        ).splitlines()    )print(total)
```

# 8. 创建目录

## 创建一级目录

```
from pathlib import PathPath("output").mkdir()
```

---

## 创建多级目录

```
Path("output/2026/report").mkdir(    parents=True,    exist_ok=True)
```

相当于：

```
mkdir -p output/2026/report
```

# 9. 删除文件

```
from pathlib import PathPath("temp.txt").unlink()
```

---

## 忽略不存在

```
Path("temp.txt").unlink(    missing_ok=True)
```

相当于：

```
rm -f temp.txt
```

```

```


# 12. CLI 项目中的典型写法

你现在的项目很适合这样改：

```
from pathlib import Pathimport sysfile_path = Path(args.filename)if not file_path.exists():    sys.exit("File not found")text = file_path.read_text(    encoding="utf-8")
```

甚至：

```
if not file_path.is_file():    sys.exit("Not a file")
```

这样比：

```
try:    with open(fname):        ...except FileNotFoundError:
```

更符合现代 Python 风格。

# 跨平台最容易踩的坑

### ❌ 不要手写路径分隔符

错误：

```
path = "data\\file.txt"
```

或者：

```
path = "data/file.txt"
```

正确：

```
path = Path("data") / "file.txt"
```

---

### ❌ 不要依赖当前工作目录

错误：

```
Path("config.json")
```

因为程序可能从别的目录启动。

更稳：

```
BASE_DIR = Path(__file__).parentconfig = BASE_DIR / "config.json"
```

---

### ❌ 不要用字符串判断路径

错误：

```
if ".txt" in filename:
```

正确：

```
path = Path(filename)if path.suffix == ".txt":    ...
```
