## 概念及代码

`str.capitalize()`[](https://docs.python.org/zh-cn/3.14/library/stdtypes.html#str.capitalize "Link to this definition")
返回原字符串的副本，其首个字符大写，其余为小写

`str.casefold()`
返回原字符串消除大小写的副本。消除大小写的字符串可用于忽略大小写的匹配。

消除大小写类似于转为小写但是更为彻底因为它会移除字符串中的所有大小写变化形式。 例如，德语小写字母 `'ß'` 相当于 `"ss"`。 由于它已经是小写了，[`lower()`](https://docs.python.org/zh-cn/3.14/library/stdtypes.html#str.lower "str.lower") 不会对 `'ß'` 做改变；而 [`casefold()`](https://docs.python.org/zh-cn/3.14/library/stdtypes.html#str.casefold "str.casefold") 会将其转为 `"ss"`。 例如：


```python
>>>'straße'.lower()
'straße'
>>>'straße'.casefold()
'strasse'
```

`str.center(width,fillchar=' ',/)`
返回长度为width的局中字符串。使用指定的fillchar进行填充空位(默认为ASCII空格)
如果width小于等于len(s)则返回原来的字符串
```python
>>>'Python'.center(10)
'  Python  '
>>>'Python'.center(10,'-')
'--Python--'
>>>'Python'.center(4)
'Python'
```

## 去除空白字符`.strip()`

去除首尾空白：

```python
s = "  hello \n"print(s.strip())
```

输出：

```
hello
```

---

```python
.rstrip()   # 去除右边空白.lstrip()   # 去除左边空白
```

# 字符串分割（重点）

## `.split()`

按空格分割：

```python
s = "I love python"print(s.split())
```

输出：

```
['I', 'love', 'python']
```

---

## 按指定字符分割

```python
s = "a,b,c"print(s.split(","))
```

输出：

```
['a', 'b', 'c']
```
---
# 字符串拼接

## `.join()`

```python
words = ["I", "love", "python"]s = " ".join(words)print(s)
```

输出：

```
I love python
```
---
# 字符串替换

## `.replace()`

```
s = "I like Java"print(s.replace("Java", "Python"))
```

输出：

```
I like Python
```
