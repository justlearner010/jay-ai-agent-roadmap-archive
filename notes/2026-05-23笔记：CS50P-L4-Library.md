# `sys.argv`（非常重要）

这是 Lecture 4 后半部分重点。

---

## 什么是 argv

`argv`：

- argument vector
- 命令行参数

---

## 示例

```
import sysprint(sys.argv)
```

运行：

```
python hello.py David
```

输出：

```
['hello.py', 'David']
```

含义：

|元素|含义|
|---|---|
|argv[0]|脚本名|
|argv[1]|第一个参数|

---

## 常见错误

如果：

```
print(sys.argv[1])
```

但没传参数：

```
python hello.py
```

会报：

```
IndexError
```

于是课程引入：

```
if len(sys.argv) < 2:    sys.exit("Too few arguments")
```
