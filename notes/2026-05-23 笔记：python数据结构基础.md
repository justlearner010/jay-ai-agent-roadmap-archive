## 代码例子

### 列表

```python
list.append() 
在列表的末尾添加一项
list.extend()
通过添加来自iterable的所有项来做扩展列表，
list.insert(index,value)
在指定位置插入元素，第一个参数是插入元素的索引
list.remove()
移除列表的第一个元素
list.clear()
移除列表的所有项 类似于 del a[:]
list.index()
返回列表中value首次出现所在的从零开始的索引。
list.count()
返回列表中value的出现次数
list.reverse()
反转列表的元素
list.copy()
返回列表的浅拷贝
list.sort(*,key=None,reverse= False)
原地排序列表中的元素
```

### 字典

python中的字典相当于C++中的`std::unordered_map`

字典包含键值对<k,v>，可以通过key来查找value，顾名思义就和我们用的字典一样
```python
# 创建字典
dic = {'Chinese':100,'English':50,'Japanese':80}
dic['French'] = 25

>>> dic['Chinese']
100
>>> dic['Germany']
Traceback (most recent call last):
  File"<stdin>",line 1,in <module>
KeyError:'Germany'
>>>print(dic.get('Germany'))
None
>>>del dic['Japanese']
>>>dic['Germany'] = 12
>>>dic
{'Chinese':100,'English':50,'French':25,'Germany':12}
>>>list(dic)
['Chinese','English','French','Germany']
>>>sorted(dic)
['Chinese','English','French','Germany']#按照key的字符串顺序排序
>>>'Chinese' in dic
True
>>>'Japanese' in dic
False
```


### `dict()`构造函数的使用
```python
>>>dict([('Chinese',100),('English',50),('French',25),('Germany',12)])
```

### 循环的技巧

`items()`:这个方法可以返回字典的键值
```python 
for k,v in dic.items():
	print(f"There are{v} {k} books")
>>>There are 100 Chinese books
>>>There are 50 English books
>>>There are 25 French books
>>>There are 12 Germany books
```
