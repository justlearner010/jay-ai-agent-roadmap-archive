### 1. 编程入门与环境

- **Python 编程环境**：使用 VS Code（或任何文本编辑器）来编写 Python 代码。
- **命令行界面 (CLI)**：通过终端运行 Python 程序，而不是点击图标。
- **程序文件**：Python 文件以 `.py` 结尾，保存代码供解释器运行。
- **Python 解释器**：负责将 Python 代码翻译为计算机可执行的二进制指令。

---

### 2. 第一个程序示例

- **Hello, World 程序**：
    
    ```
    print("hello, world")
    ```
    
- **概念**：
    - **函数 (Function)**：程序中的动作或操作，例如 `print()`。
    - **参数 (Argument)**：输入到函数中以影响其行为，例如 `"hello, world"`。
    - **副作用 (Side Effect)**：函数对外界产生的可见影响，如在屏幕上显示文字。

---

### 3. 用户输入与变量

- **input() 函数**：获取用户输入并返回为字符串。
- **变量 (Variable)**：用于存储值，可重用。
    - 赋值用 `=`，右侧值存入左侧变量。
- **示例**：
    
    ```
    name = input("What's your name? ")print("Hello,", name)
    ```
    

---

### 4. 字符串操作与拼接

- **字符串 (String)**：文本类型，Python 中称为 `str`。
- **拼接 (Concatenation)**：
    - 使用 `+` 将字符串连接。
    - 也可以使用多个参数传入 `print()`，逗号自动插入空格。
- **格式化输出**：
    - 使用 `{}` 和 `.format()` 或 f-string（课后延伸）实现更简洁的输出。
- **转义字符 (Escape)**：
    - 例如 `\"` 用于在字符串中显示引号。
    - `\n` 表示换行。

---

### 5. 注释与伪代码

- **注释 (Comment)**：
    - 单行：`#` 开头。
    - 多行：三引号 `""" ... """` 或多行 `#`。
    - 供程序员或同事理解代码用途，计算机忽略。
- **伪代码 (Pseudocode)**：
    - 使用自然语言描述程序逻辑。
    - 有助于分解任务，规划代码实现步骤。

---

### 6. 错误与调试

- **语法错误 (Syntax Error)**：如缺少括号或引号。
- **调试技巧**：
    - 阅读错误信息。
    - 按照提示修复。
    - 注释和逐行检查程序有助于定位问题。

---

### 7. 函数参数类型

- **位置参数 (Positional Parameters)**：按顺序传入函数的值。
- **命名参数 (Named Parameters)**：可以通过参数名指定，改变默认行为。
    - `print(..., sep=' ', end='\n')`：
        - `sep`：多参数间的分隔符，默认空格。
        - `end`：行尾符号，默认换行 `\n`。

---

- 学习了基本的 Python 编程流程：
    - 编写代码 → 使用解释器运行 → 处理输入输出 → 调试。
- 介绍了核心概念：
    - **函数、参数、变量、字符串、注释、输入输出、副作用**。
- 掌握了初步的 **程序逻辑设计与调试技巧**，为后续复杂程序打基础。
