
结合你的情况（有 C、Python 基础，准备学 Next.js 和 AI Agent），我建议先掌握 **20% 的 TS 语法，解决 80% 的开发需求**。

---

# 第一部分：变量

## JS

```js
let age = 19;
const name = "Kris";
```

---

## TS

```ts
let age: number = 19;

const name: string = "Kris";
```

不过大部分时候可以省略：

```ts
const age = 19;
```

TS 会自动推导：

```ts
age: number
```

---

# 第二部分：基础类型

```ts
let age: number = 19;

let name: string = "Kris";

let passed: boolean = true;
```

---

数组：

```ts
let scores: number[] = [90, 85, 100];
```

---

字符串数组：

```ts
let books: string[] = [
    "GEB",
    "CSAPP"
];
```

---

# 第三部分：函数

Python：

```python
def add(a: int, b: int) -> int:
    return a + b
```

---

TS：

```ts
function add(
    a: number,
    b: number
): number {
    return a + b;
}
```

---

箭头函数：

```ts
const add = (
    a: number,
    b: number
): number => {
    return a + b;
};
```

AI项目里非常常见。

# 第五部分：type

你刚学的重点。

```ts
type User = {
    name: string;
    age: number;
};
```

使用：

```ts
const user: User = {
    name: "Kris",
    age: 19
};
```

# 第六部分：联合类型

限制取值。

```ts
type SummaryMode =
    | "brief"
    | "bullets";
```

---

```ts
function summarize(
    text: string,
    mode: SummaryMode
) {
}
```

只能传：

```ts
"brief"
```

或者：

```ts
"bullets"
```

# 第七部分：数组类型

```ts
type Message = {
    role: string;
    content: string;
};
```

---

单个：

```ts
const msg: Message = {
    role: "user",
    content: "hello"

```

---

多个：

```ts
const messages: Message[] = [
    msg

```

这是 Agent 开发里最常见的数据结构之一。

# 第八部分：可选字段

```ts
type User = {
    name: string;
    age?: number;
};
```

问号表示：

```ts
age
```

可以没有。

---

合法：

```ts
const user: User = {
    name: "Kris"
};
```

# 第九部分：interface


# 第十部分：泛型（最重要）

这是 TS 最大的价值之一。

例如：

```ts
function identity<T>(
    value: T
): T {
    return value;
}
```

---

调用：

```ts
identity<string>("hello");

identity<number>(123);
```

---

数组泛型：

```ts
const books: Array<string> = [
    "GEB",
    "CSAPP"
];
```

---

以后你会经常见到：

```ts
Promise<string>

Promise<User>

Record<string, number>
```

全是泛型。

---

# 第十一部分：异步

现代前端和 AI 项目必学。

```ts
async function getData() {
    return "hello";
}
```

---

调用：

```ts
const result = await getData();
```

---

AI Agent 项目里：

```ts
const response =
    await openai.chat.completions.create(...);
```

天天写。

# 第十二部分：class

```ts
const s = new Student("Kris");
```

```ts

```

而不是：

```ts
class
```

# AI Agent开发最常见的语法

如果你接下来学：

- TypeScript

- Node.js

- Next.js

- OpenAI SDK

- LangGraph


你会发现出现频率最高的是：

```ts
type
```

```ts
async / await
```

```ts
function
```

```ts
Promise<T>
```

```ts
数组
```

```ts
对象
```

```ts
联合类型 |
```

比如一个典型 Agent 状态：

```ts
type AgentState = {
    messages: Message[];
    currentTask: string;
    status:
        | "idle"
        | "running"
        | "finished";
};
```

如果这些语法你能熟练写出来，已经足够开始做 AI Agent 项目了。后面的高级类型（Mapped Types、Conditional Types、Infer 等）可以等项目里遇到再学。
