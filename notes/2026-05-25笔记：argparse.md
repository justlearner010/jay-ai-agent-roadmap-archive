## 概念及代码
`argparse` 是 Python 标准库里专门用来“解析命令行参数（command-line arguments）”的模块。

`python app py input.txt 100`
- app.py是程序
- input.txt
- 100
这后面的内容就是命令行参数

`argparse`的作用是：
- 自动读取这些参数
- 帮你转换类型
- 自动生成 --help
- 检查参数是否合法
- 给CLI（命令行工具）提供专业接口


### 例子
以昨天的作业为例：
```python
import sys

fname = sys.argv[1]
chunk_size = sys.argv[2]
```
使用argparse改写：
```python
import argparse

parser = argparse.ArgumentParser()

parser.add_argument("fname")
parser.add_argument("chunk_size")

args = parser.parse_args()

print(args.fname)
print(args.chunk_size)
```


### 比`sys.argv`强在哪里？

#### 1.参数缺失会直接报错

`IndexError`
但是`argparse`会:
`usage:app.py[-h] fname chunk_size`

#### 2.自动类型检查
```python
type = int
>>>python app.py test.txt abc
invalid int value
```


#### 3.自动生成帮助文档
你写：`parser = argparse.ArgumentParser(description="文本切块程序")`
然后:`python app.py --help`
自动生成:
```python
usage: app.py [-h] fname chunk_size

文本切块程序

positional arguments:
  fname
  chunk_size

options:
  -h, --help
```


# add_argument() 是核心

比如：

```
parser.add_argument("fname")
```

表示：

必须参数。

---

而：

```
parser.add_argument("--size")
```

表示：

可选参数。
