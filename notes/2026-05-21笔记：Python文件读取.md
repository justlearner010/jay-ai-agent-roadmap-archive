### Part1：Python文件读取

文本文件和二进制文件

对于文本文件的理解
```python 打开文本文件
tf = open("f.txt","rt")
print(tf.readline())
# 从文本中读出一行
tf.close()
```

```python 
# 读取二进制文件
bf = open("f.txt","rb")
print(bf.readline())
bf.close()
```

UTF-8编码

```python
filename = 'test.txt'
with open(filename)as file_object:
	content = file_object.read();
```
**with open**语句是确定文件打开的语句，它保证了文件会适时打开和关闭，不使用`open()` `close()`来控制文本的打开关闭是因为如果`close()`没有正常执行的话文本将不会关闭

### 文件处理过程

打开->操作->关闭

`<变量名> = open("文件名","打开格式")`


### 打开模式
`r`:read-only,return FileNotFoundError if not exist
`w`:write and cover if exist ,create and write if not exist
`x`:write if not exist,else FileExistError
`a`:add if exist,else create a new file

默认只读模式，文本形式
rt：read only and txt
a+:add after the txt


### 读取模式

`.read`:读入字符
`.readline`：读入一行
`.readlines`：读入多行

### 全文的操作
open() -> .read()->close()

对于大文件：
按照数量读入`
```python
fname = input("filename")
fo = open(fname,"r")
txt = fo.read(2)

while (txt !=""):
	txt=fo.read(2)
fo.close()
```
如果没有读取完的话继续读取，但是分阶段读取，这种思路可以考虑

按行存储：
```python
fname = input("filename")
fo = open(fname,"r")
for line in fo.readlines():
	print(line)
fo.close()

```
